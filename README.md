# 📓 Cuaderno Interactivo — Movimiento Oscilatorio

> **Proyecto académico interactivo** desarrollado para la clase de Física  
> Universidad Tecnológica de Pereira (UTP) — Colombia

---

## 📌 Descripción

Este repositorio contiene un **cuaderno interactivo web** sobre el tema de **Movimiento Oscilatorio**, que abarca conceptos fundamentales de la física como el movimiento armónico simple (MAS), péndulos, sistemas masa-resorte, oscilaciones amortiguadas y forzadas, entre otros.

El cuaderno está diseñado para ser una herramienta educativa dinámica, con visualizaciones, simulaciones y explicaciones interactivas que facilitan la comprensión del movimiento oscilatorio a nivel universitario.
