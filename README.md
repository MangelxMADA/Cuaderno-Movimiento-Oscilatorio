# Fisica 3 - Universidad Tecnologica de Pereira

Apuntes interactivos de Fisica 3 con simulaciones, ecuaciones y contenido en formato IEEE.

## Contenido

- Temas de clase interactivos
- Simulaciones computacionales
- Visor de archivos LaTeX y PDFs 
- Guías y talleres resueltos

## Despliegue

Este repositorio está estructurado para ser desplegado fácilmente utilizando [GitHub Pages](https://pages.github.com/).
El archivo principal de la aplicación es `index.html` ubicado en la raíz del repositorio.
